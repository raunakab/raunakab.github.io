# Personal Website + Blog

Structure of the project:
```
/
|- content
    |- pages
        # Contains information about myself
        |- about.md

    # All blog entries
    |- ${YYYY}-{MM}-{DD}-{NAME1}.md
    |- ...
    |- ${YYYY}-{MM}-{DD}-{NAMEn}.md

    # content root
    |- _index.md

|- themes
    |- lightspeed
        |- # All html + styles required for this theme
        |- ...
```

Based off of the [`lightspeed`](https://www.getzola.org/themes/lightspeed).
